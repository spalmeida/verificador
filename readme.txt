== Descrição ==
Este plugin retorna uma tabela com a versão atual e a última versão dos plugins e temas (ativos e inativos) no WordPress e verifica se há atualizações diretamente do GitHub.

== Instalação ==
1. Faça upload da pasta para `wp-content/plugins/` ou baixe diretamente do GitHub.
2. Ative o plugin no painel de controle do WordPress.

== Atualizações ==
Este plugin verifica e atualiza diretamente do GitHub. Use a versão mais recente no repositório GitHub.
